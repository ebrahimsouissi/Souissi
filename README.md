# Souissi Ibrahim — Portfolio Website

Personal portfolio and freelancer site for Souissi Ibrahim, an independent graphic designer based in Tunis.

This is **Phase 2** of the project: the public-facing site, built as static HTML/CSS. Phase 3 will convert this into a Next.js application with an admin panel and database.

---

## Pages

- `index.html` — Homepage (hero, featured work, about strip, services, testimonials, CTA)
- `work.html` — Full portfolio grid with category filters
- `project.html` — Individual project case study (template; one per project in Phase 3)
- `about.html` — Bio, portrait, skills, resume download
- `services.html` — Detailed services with starting-price tiers and process
- `contact.html` — Inquiry form with project type, budget, and timeline fields
- `styles.css` — Shared stylesheet for all pages

---

## How to view locally

No build tools needed for Phase 2 — just open the files in a browser.

1. Download or clone this repository
2. Make sure all 7 files are in the same folder
3. Double-click `index.html` to open in your browser
4. Use the navigation to move between pages

---

## Design system

- **Typography**: Fraunces (display serif) + Geist (sans) + Geist Mono (mono accents)
- **Palette**: Near-black background `#0e0e0c`, off-white text `#f4f1ea`, coral accent `#ff4d3d`
- **Aesthetic**: Editorial / magazine-inspired, dark mode, asymmetric grids
- **Motion**: Scroll-triggered reveals, hover micro-interactions, staggered hero entry
